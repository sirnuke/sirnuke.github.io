SteamBridge
A set of libraries and tools that enable Linux Steam users to run Windows applications in their library without the Windows Steam client.
It comes in the form of a replacement steam_api.dll library, that proxies between Windows and Linux function calls, a Winelib library that proxies between Wine and the native Linux libsteam_api.so, and a set of Python and shell scripts to manage everything.
